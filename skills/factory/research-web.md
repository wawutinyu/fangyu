# research-web

外证优先可复核来源；本地对照避免空话。

## 步骤

1. **定问题**：一句话写清要查什么（版本、API、对比项）。
2. **检索**：`websearch` 拿候选标题与 URL；挑 1～3 个权威源。
3. **拉页**：`webfetch` 读正文；摘录可核对的句子/表格要点（带 URL）。
4. **对照本地**（可选）：`glob` / `grep` / `read` 看仓内是否已有同类实现或配置。
5. **收口**：结论 = 来源 URL + 要点 +（若有）本地路径对照；不要改文件。

## 委派

父 Agent 可用 `task` → `scout`；或主环加载本技能后自调 web 工具。

## 反例

- 无 URL 的「据我所知」
- 只搜不 fetch 就长篇总结
- 把外网结论直接写成仓内事实却不核对
