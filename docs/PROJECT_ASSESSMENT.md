## 最值钱的地方

1. **北极星可验收** — 搭→导出→离 Studio→受约束→可测，不是口号。  
2. **机床 + 原料 + 门禁同在** — loop / materials / `factory_gate` 形成最小工厂闭环。  
3. **宪法随包执行** — 差异化前提已落地（P0-1）。  
4. **观 + A2A 值班** — L2 互操作有人机现场，不只 JSON。  
5. **测试文化** — 说出去的能力大多有单测/gate 钉住。

## 总结

胜负手仍是：**导出物在包内自洽地多轮干活，且门禁 live 能绿。**  
观与跨厂值班是正确的 L2 投资，但不应再替代 G1 稳定性与总毕业缺口。  
旧评估「~6 产线雏形」→ 上调为 **~7.2 可演示产线**；总毕业口径保持 **~5.5 / ~65%**，避免自嗨。P0–P2/P4 已落代码；P3 真 IM 仍是最大体验缺口（刻意暂缓）。
